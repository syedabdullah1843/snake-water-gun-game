# 🐍 Snake Water Gun Game (Python)

A simple command-line game built using Python where you play **Snake-Water-Gun** against the computer.

---

## 🎮 Game Rules

* 🐍 Snake drinks Water → Snake wins
* 💧 Water douses Gun → Water wins
* 🔫 Gun kills Snake → Gun wins

---

## 🚀 Features

* 🤖 Random computer choice
* 🎯 User input-based gameplay
* ⚡ Instant result (Win/Lose/Draw)
* 💻 Simple and beginner-friendly

---

## 🛠 Tech Stack

* Python 3
* Built-in module: `random`

---

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/snake-water-gun.git
cd snake-water-gun
```

2. No external libraries required ✅

---

## ▶️ How to Play

Run the script:

```bash
python game.py
```

Enter your choice when prompted:

```
s → Snake
w → Water
g → Gun
```

---

## 🧠 Example Output

```
Enter your choice: s
Player chose Snake
Computer chose Water
player win!
```

---

## ⚠️ Note

* Make sure to enter only:

  * `s` for Snake
  * `w` for Water
  * `g` for Gun
* Invalid input may cause an error
